QAIYRYM (Қайырым) — это децентрализованный протокол доверия (Trustless Charity Protocol), решающий проблему «черного ящика» в благотворительности. Мы переносим учет социальной помощи из бумажных журналов в неизменяемый реестр Solana.

🛡 ТЕХНОЛОГИЧЕСКИЙ СТЕК «TETRA-SHIELD» (УГЛУБЛЕННО):
OFF-CHAIN VALIDATION (Источник Истины):

Интеграция со школьными социальными педагогами.

Данные верифицируются через государственные базы (социально-уязвимые слои населения).

Это исключает «цифровое попрошайничество» — помощь получают только те, чья нужда подтверждена оффлайн.

AI-ORACLE (Глаза Протокола — Gemini 2.0 Flash):

Computer Vision: Анализ фотографий передачи помощи. Бот распознает товары, сверяет их с чеками и подтверждает факт покупки.

Metadata Audit: Автоматическая сверка EXIF-данных (GPS-координаты и временные метки должны совпадать с адресом семьи).

Fraud Detection: Выявление дубликатов фото или поддельных отчетов.

SOLANA ON-CHAIN LOGGING (Слой Прозрачности):

Каждое действие (поступление заявки -> покупка -> передача) хэшируется.

Хэш записывается в Solana Devnet (с переходом в Mainnet).

Результат: Любой меценат может через Explorer проверить, что его помощь дошла, увидев хэш транзакции, привязанный к отчету.

FEEDBACK LOOP (Кольцо Верификации):

Получатель подтверждает получение через уникальный одноразовый код (OTP) в боте.

Без подтверждения получателя волонтер не получает баллы в рейтинг.

# QAIYRYM: ЦИФРОВАЯ ЭКОСИСТЕМА ДОВЕРИЯ (БАЗА ЗНАНИЙ v1.0)

## 1. СТРАТЕГИЧЕСКИЙ СТЕК (ДЛЯ ПАРТНЕРОВ И АКМАТА)
- Технология контроля: AI-оркестрация (каскад ИИ-агентов).
- Слой верификации: Тройной аудит (Фото + GPS + Подтверждение получателя).
- Блокчейн-протокол: Solana (Devnet). Каждое действие хешируется (SHA-256).
- Инструмент доверия: SBT (Soulbound Tokens) — непередаваемые токены репутации для волонтеров.
- ESG-эффект: 100% прозрачность целевого использования средств. Автоматическая генерация отчетов для бизнеса.

## 2. КРЫЛЬЯ ПРОЕКТА (ОПЕРАЦИОНКА)
- [СИНЕЕ КРЫЛО]: Социалка. Помощь маломобильным гражданам, многодетным семьям, доставка продуктов.
- [БЕЛОЕ КРЫЛО]: Культура. Организация мероприятий, домбра-классы, сохранение традиций.
- [ФИОЛЕТОВОЕ КРЫЛО]: Развитие. IT-обучение, воркшопы по ИИ, хакатоны, лидерство.

## 3. ДЛЯ ВОЛОНТЕРОВ (МОТИВАЦИЯ И ГЕЙМИФИКАЦИЯ)
- Статус "Candidate": После регистрации. Доступ к легким задачам.
- Статус "Verified Member": После 3-х успешно закрытых и проверенных ИИ заявок.
- Награда: За каждый закрытый кейс — запись в SBT-паспорт (цифровое резюме для будущего трудоустройства).
- Координатор: Каждые 10 волонтеров закреплены за опытным ментором.

## 4. ДЛЯ ПОЛУЧАТЕЛЕЙ ПОМОЩИ (БЕЗОПАСНОСТЬ)
- Срок обработки заявки: ИИ-матчинг находит волонтера за 5-15 минут.
- Безопасность: Все волонтеры проходят проверку личности.
- Контроль качества: Получатель ставит финальную оценку в боте после получения помощи.

## 5. ЭКОНОМИКА (ДЛЯ ИНВЕСТОРОВ)
- Монетизация: Подписка на ESG-дашборд для корпораций.
- HR-ценность: Платный доступ для рекрутеров к базе "Verified Leaders" (топ-волонтеров с подтвержденными навыками).
- Партнеры: IT-HUB Актобе, Школы города, Грантовая поддержка (в процессе).
